# ВЕДЬМАК

[Данный сайт](https://konstantin333.github.io/the-witcher/ "Ведьмак") был разработан в первую очередь для фанатов фэнтези вселенных. Одной из которых как раз и является Ведьмак - книжная франшиза, которая обретя популярность, превратилась в мировую медиафраншизу, состоящую из книг, игр и сериалов.

На этом сайте вы можете ознакомиться с:

* кратким описанием вселенной Ведьмака и её главными героями;

* книгами Анджея Сапковского и сайтами, на которых их можно приобрести;

* играми от разработчиков CD Project RED и музыкой из них;

* сериалом от Netflix;

* и небольшим бонусом в виде персонажей и монстров из бестиария.

_P.S. Сайт адаптирован для просмотра как на ПК, так и на планшетах и мобильных устройствах._
